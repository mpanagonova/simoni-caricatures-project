# Test Summary Report – Simoni Caricatures Website

## 1. Overview

This document summarizes the results of the manual testing performed on the Simoni Caricatures website.

The testing covered key user flows, including navigation, forms, gallery functionality, and responsiveness.

---

## 2. Test Execution Summary

- Total Test Cases: 20
- Executed: 20
- Not Executed: 0

---

## 3. Test Results

- Passed: 16
- Failed: 4
- Not Run: 0

---

## 4. Defect Summary

A number of defects and usability issues were identified during testing.

### Key Defects:

1. Language selector contains only one option
2. UI alignment issues in gallery  
3. Missing gallery filtering functionality  
4. No option to edit/delete comments  
5. Missing account deletion functionality  

---

## 5. Test Coverage

The following areas were tested:

- Core website functionality (homepage, navigation)
- Gallery functionality (image loading, navigation, sorting)
- Forms (contact form, order form)
- Blog (comments functionality)
- User account (basic functionality)
- Responsiveness (desktop and mobile)
- Accessibility (basic keyboard navigation)

---

## 6. Observations

- The website performs well in terms of basic navigation and functionality
- Several usability gaps were identified
- Some UI inconsistencies affect user experience

---

## 7. Risks & Limitations

- No access to backend systems or database
- Testing was limited to front-end behavior (black-box testing)
- Performance and security testing were not included

---

## 8. Conclusion

The website is functional but contains several usability and UI inconsistencies that should be addressed.

The testing process demonstrated a structured QA approach, including test planning, execution, and defect reporting.
